# Meta-Constraints & Ground Rules from Spec

Rules and guidance from the "Important Notes" section of the [[raw/Software Engineering Take Home Test v2.0.pdf|take-home spec]].

## Assumptions

- You can make assumptions on any information not specified in the document
- Additional code libraries or dependencies may be used as necessary

## AI Usage

- AI coding assistants and tools are permitted
- You are expected to check the correctness of the solution and be familiar with the code

## Source Control

- Use Git to manage and track changes of your source code

## Timeline

- Complete within 3 calendar days of receiving the test
- Can request up to 2 additional days by emailing the team

## Team Context

This is for the **xDigital AI Products Team** at HTX. The role is a full-stack software engineer developing web applications integrated with AI capabilities. Expected knowledge areas:
- Frontend: JavaScript/TypeScript, React
- Backend: JavaScript/TypeScript, Node.js, Python
- Database: PostgreSQL
- GenAI: LLM capabilities
